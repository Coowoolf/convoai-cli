# Embedded API Keys: Built-in LLM/TTS for Zero-Config Experience

**Date:** 2026-04-13
**Version:** v1.9.0
**Status:** Draft

## Overview

Embed Qwen (LLM) and MiniMax (TTS) API keys into the CLI so developers can get a voice agent running without configuring any provider keys. Keys are encrypted at publish time and never appear in Git history. ASR uses Agora's built-in Ares (no key needed).

## Goals

1. Zero-key quickstart: developer selects "built-in" provider → skips API key input → works immediately
2. Keys never appear in source code, Git history, or plaintext in npm package
3. Developers can still configure their own keys for any provider
4. Graceful fallback when embedded keys are unavailable (dev environment, removed in future)

## Non-Goals

- Backend proxy for key management (too much infrastructure for demo keys)
- Key rotation without CLI release (accept this limitation)
- Protecting against determined reverse engineering (only protecting against casual discovery)
- Embedding ASR keys (Ares is Agora-native, no key needed)

## Build-Time Injection

### Flow

```
Your machine (publish):
  1. Environment variables: CONVOAI_QWEN_KEY, CONVOAI_MINIMAX_KEY
  2. npm publish → prepublishOnly triggers embed-keys script
  3. Script reads env vars → XOR encrypts with SHA-256 derived salt → writes embedded.ts
  4. tsc compiles → dist/keys/embedded.js (encrypted hex blobs)
  5. npm uploads package (encrypted blobs, no plaintext)

Developer machine (clone):
  - No embedded.ts (in .gitignore)
  - getEmbeddedKey() returns null
  - "Built-in" options in quickstart will error with helpful message

End user machine (npm install):
  - dist/keys/embedded.js exists (encrypted)
  - getEmbeddedKey() decrypts at runtime
  - "Built-in" options work seamlessly
```

### Encryption

XOR cipher with SHA-256 derived key:
- Salt: `sha256('convoai-embedded-v1')` → 32 bytes
- Each key byte XORed with salt byte (cycling)
- Output: hex string stored in generated file
- Not cryptographically secure, but not greppable/scannable

### Security Matrix

| Attack Vector | Protected? |
|---------------|-----------|
| GitHub search / grep source | Yes — key never in git |
| `npm pack` then grep for key pattern | Yes — only encrypted hex |
| `strings` on dist/ binaries | Yes — XOR removes readable patterns |
| Read decryption code + manually execute | No — determined reverse engineering |
| Runtime interception (console.log, debugger) | No |

## Resolve Module: `src/keys/resolve.ts`

Hand-written, committed to git. Contains decryption logic and the `resolveEmbeddedKeys` function.

```typescript
import { createHash } from 'node:crypto';

function decrypt(hex: string): string {
  const data = Buffer.from(hex, 'hex');
  const salt = createHash('sha256').update('convoai-embedded-v1').digest();
  return Buffer.from(data.map((b, i) => b ^ salt[i % salt.length])).toString();
}

export function getEmbeddedKey(provider: 'qwen' | 'minimax'): string | null {
  try {
    // Dynamic import — file only exists in published package
    const { EMBEDDED_KEYS } = require('./embedded.js');
    const hex = EMBEDDED_KEYS[provider];
    if (!hex) return null;
    return decrypt(hex);
  } catch {
    return null;
  }
}
```

### Runtime Resolution

Before sending API requests, check for the `__embedded__` placeholder and replace with the real key:

```typescript
export function resolveEmbeddedKeys(config: {
  llm?: Record<string, unknown>;
  tts?: Record<string, unknown>;
}): void {
  if (config.llm?.api_key === '__embedded__') {
    const key = getEmbeddedKey('qwen');
    if (key) {
      config.llm.api_key = key;
    } else {
      throw new Error('Built-in Qwen key not available. Run "convoai quickstart" to configure your own API key.');
    }
  }
  if (config.tts?.params && (config.tts.params as Record<string, unknown>).key === '__embedded__') {
    const key = getEmbeddedKey('minimax');
    if (key) {
      (config.tts.params as Record<string, unknown>).key = key;
    } else {
      throw new Error('Built-in MiniMax key not available. Run "convoai quickstart" to configure your own API key.');
    }
  }
}
```

Call sites: `go.ts`, `quickstart.ts`, `agent/start.ts`, `phone/go.ts`, `phone/send.ts` — resolve before sending the API request.

## Quickstart Integration

### LLM Vendor Selection

Add "Qwen (built-in)" as first choice in the LLM vendor prompt:

```
? Select LLM provider:
  ❯ Qwen (built-in, no API key needed)
    OpenAI GPT-4o
    OpenAI GPT-4o-mini
    Anthropic Claude
    Google Gemini
    DeepSeek
    ...
```

When selected:
- Set `llm.vendor` = `'qwen'`
- Set `llm.model` = appropriate Qwen model
- Set `llm.api_key` = `'__embedded__'`
- Skip the "Enter your API key" prompt
- Set `llm.style` = `'openai'` (Qwen uses OpenAI-compatible API)

### TTS Vendor Selection

Add "MiniMax (built-in)" as first choice:

```
? Select TTS provider:
  ❯ MiniMax (built-in, no API key needed)
    Microsoft
    ElevenLabs
    ...
```

When selected:
- Set `tts.vendor` = `'minimax'`
- Set `tts.params.key` = `'__embedded__'`
- Skip key input

### ASR Vendor Selection

Add/promote "Agora Ares" as first choice:

```
? Select ASR provider:
  ❯ Agora Ares (built-in, no key needed)
    Deepgram
    Google
    ...
```

When selected:
- Set `asr.vendor` = `'ares'` (or whatever the Agora vendor name is)
- No key needed, no prompt

### Full Zero-Key Path

If user selects all three built-in options, they skip ALL key inputs in quickstart. The only required credentials remain `app_id`, `customer_id`, `customer_secret` (Agora platform credentials).

## Error Handling

| Scenario | Behavior |
|----------|----------|
| Dev environment, no embedded.ts | `getEmbeddedKey()` returns null; selecting "built-in" in quickstart works (saves `__embedded__`), but `convoai go` throws clear error: "Built-in key not available. Configure your own." |
| Embedded key expired/revoked | API call fails with provider error; CLI shows the error as-is |
| User previously configured own key | No change — `__embedded__` placeholder only set by quickstart built-in selection |
| User switches from built-in to own key | `convoai config set llm.api_key sk-xxx` overwrites the placeholder |
| `--json` mode with missing embedded key | JSON error output |

## Embed Script: `scripts/embed-keys.ts`

```typescript
#!/usr/bin/env tsx
import { createHash, randomBytes } from 'node:crypto';
import { writeFileSync } from 'node:fs';
import { join } from 'node:path';

const SALT = createHash('sha256').update('convoai-embedded-v1').digest();

function encrypt(plaintext: string): string {
  const data = Buffer.from(plaintext);
  const encrypted = Buffer.from(data.map((b, i) => b ^ SALT[i % SALT.length]));
  return encrypted.toString('hex');
}

const keys: Record<string, string> = {};

const qwen = process.env.CONVOAI_QWEN_KEY;
const minimax = process.env.CONVOAI_MINIMAX_KEY;

if (!qwen || !minimax) {
  console.error('Missing environment variables: CONVOAI_QWEN_KEY and/or CONVOAI_MINIMAX_KEY');
  console.error('Set them before publishing. Skipping key embedding.');
  // Write empty file so TypeScript compiles
  writeFileSync(
    join(__dirname, '..', 'src', 'keys', 'embedded.ts'),
    '// No keys embedded (missing env vars)\nexport const EMBEDDED_KEYS: Record<string, string> = {};\n',
  );
  process.exit(0);
}

keys.qwen = encrypt(qwen);
keys.minimax = encrypt(minimax);

const content = `// Auto-generated by scripts/embed-keys.ts — do not edit or commit
export const EMBEDDED_KEYS: Record<string, string> = ${JSON.stringify(keys, null, 2)};
`;

writeFileSync(join(__dirname, '..', 'src', 'keys', 'embedded.ts'), content);
console.log('✓ Embedded keys written to src/keys/embedded.ts');
```

## Testing Strategy

- **Unit:** `resolve.ts` — decrypt known hex → expected plaintext; getEmbeddedKey returns null when no embedded file
- **Unit:** `embed-keys.ts` — encrypt/decrypt roundtrip with known values
- **Unit:** `resolveEmbeddedKeys` — replaces `__embedded__` placeholder; throws when key unavailable; leaves non-embedded keys untouched
- **CLI:** quickstart built-in selection skips key prompt (mock inquirer)
- **CLI:** `--help` shows updated provider lists
- **Integration:** No embedded.ts → selecting built-in → error message is helpful
