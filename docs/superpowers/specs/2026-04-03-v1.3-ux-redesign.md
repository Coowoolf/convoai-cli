# ConvoAI CLI v1.3 — UX Redesign Spec

## Goal

重新设计 quickstart 引导体验 + 新增运行时控制面板，让 ConvoAI CLI 从"能用"升级到"好看好用"。

## Scope

两个独立子系统：

1. **Quickstart UX 重构** — 新视觉、中英文分流、步骤顺序调整、引导词优化
2. **运行时控制面板** — Agent 启动后的状态面板 + 热更新参数

---

## Part 1: Quickstart UX 重构

### 视觉系统

**渐变框**：所有步骤使用蓝→粉渐变圆角框（`╭╮╰╯│`），框线颜色从 `rgb(80,150,255)` 渐变到 `rgb(220,72,195)`。

**渐变标题**：框内标题每个字用不同渐变色，配 emoji 前缀。

**进度条**：方案 1 — 渐变色块增长，右侧干净透明，显示 `N/6`。每一步进度条按比例增长。完成时进度条变绿（蓝→粉→绿渐变）。

  总览 → 项目信息 → App ID / App 证书

## Part 2: 运行时控制面板

### 主面板

Agent 启动后显示实时状态面板：

```
⚡ Agent 运行中 · A42AX... · Channel: my-room
─────────────────────────────────────────

🎙 ARES (zh-CN)        🧠 qwen-plus
🔊 MiniMax             ⏱ VAD 1000ms

对话: 3 轮 · 平均延迟 1.2s

─────────────────────────────────────────
[a] ASR  [l] LLM  [t] TTS  [v] VAD  [h] 历史  [q] 退出
```

面板每 2 秒自动刷新（对话轮数、延迟更新）。

### 快捷键交互

所有快捷键进入子菜单，子菜单有"返回主面板"选项。

#### [a] ASR 子菜单
- 切换语言
- 切换服务商

#### [l] LLM 子菜单
- 切换模型（列表选择，当前模型标注 `(当前)`）
- 切换服务商
- 调整 temperature（输入数字，显示当前值和范围）
- 调整 max_tokens
- 修改 system prompt（文本输入）

#### [t] TTS 子菜单
- 切换服务商
- 调整语速
- 调整音量

#### [v] VAD 子菜单
- 静默等待时间 silence_duration_ms（当前值 + 范围 200-3000）
- 打断检测时间 interrupt_duration_ms
- 打断模式（interrupt / append / ignore，每个带中文说明）

#### [h] 对话历史
- 显示完整对话记录
- 每条 agent 回复带延迟标注 `[1.2s]`
- 按任意键返回主面板

#### [q] 退出
- 停止 Agent
- 显示 Session Report（对话记录 + 延迟分析表格 + 摘要）

### 热更新机制

所有参数修改通过声网 ConvoAI 的 `update` API 实时生效：
- `POST /agents/{agentId}/update` 支持更新 `token`、`llm.system_messages`、`llm.params`
- 不支持运行时切换服务商（需要重启 Agent）

**因此**：
- 切换模型、temperature、max_tokens、system prompt → update API 热更新
- 切换服务商（LLM/TTS/ASR）→ 需要停止当前 Agent 并重新启动
- VAD 参数 → 需要重启 Agent

切换服务商时提示用户："需要重启 Agent，当前对话将断开。是否继续？(Y/n)"

## 技术影响

### 需要新建的文件
- `src/ui/gradient.ts` — 渐变框、渐变标题、渐变进度条的渲染函数
- `src/ui/i18n.ts` — 中英文文案数据
- `src/commands/agent/panel.ts` — 运行时控制面板

### 需要重写的文件
- `src/commands/quickstart.ts` — 全面重写 banner + step 函数 + 步骤顺序 + 引导词

### 需要修改的文件
- `src/commands/agent/join.ts` — 启动后进入控制面板
- `src/commands/agent/chat.ts` — 启动后进入控制面板
- `src/providers/catalog.ts` — 无变更（数据源不变）

### update API 限制
声网 update API 只支持更新 `token`、`llm.system_messages`、`llm.params`。不支持运行时切换 TTS/ASR/LLM vendor。切换服务商需要 stop + start。
